# 🏠 TenantRadar

**TenantRadar** is a Full-Stack Agentic AI Property Management Platform that automates tenant communication, maintenance management, and rent tracking for landlords.

The system uses an intelligent AI agent to analyze tenant messages, classify issues, create maintenance tickets, escalate urgent requests, and assist landlords through a centralized dashboard.

## 🚀 Features

✅ AI-Powered Tenant Message Processing
✅ Conversation-Based Messaging System
✅ Automated Maintenance Ticket Creation
✅ Plumbing, Electrical & HVAC Classification
✅ Priority Escalation for Urgent Issues
✅ Rent Tracking & Payment Management
✅ Property & Tenant Management
✅ AI Activity Feed & Decision Tracking
✅ Secure Authentication with NextAuth

## 🧠 AI Workflow

1. Tenant submits a message.
2. AI analyzes the content.
3. Message is classified into:

   * Maintenance Request
   * Rent Issue
   * Complaint
   * General Inquiry
4. Maintenance tickets are automatically created or updated.
5. Urgent follow-ups trigger ticket escalation.
6. Landlords monitor activities through the dashboard.
